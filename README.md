# Scorpio's Niri Dotfiles: Getting Gruvy on the Laptop

So I got a laptop not to long ago and felt constricted. I'm perfectly
fine with KDE for the most part but I wanted something fancier and 
in my opinion, more efficent. My hyprland dotfiles are a bit overkill
and currently have some setup issues so I thought to try out Niri as
I've heard it's a great WM for laptops. I'll touch up this README with 
some notes and screenshots as I get these made but the TLDR goal is
a calm and speedy work laptop. I'm using this to manage DnD Campagins,
small coding tasks, and the occasional misc task.
